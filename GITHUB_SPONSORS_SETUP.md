## Setup Instructions

1. **Go to GitHub Sponsors**: https://github.com/sponsors
2. **Click "Get sponsored"** 
3. **Complete your profile**:
   - Add profile picture
   - Write "About" section
   - Add tier descriptions (copy from this file)
   - Set payment details (Stripe Connect)
   - Submit tax information

4. **Create the tiers** (copy-paste from above):
   - Monthly: $5, $25, $100, $500
   - One-time: $10, $50, $100

5. **Add to PWA-Kit README**:
   ```markdown
   ## Support PWA-Kit
   
   PWA-Kit is open-source and free to use. If you find it valuable, please consider sponsoring development:
   
   [![Sponsor PWA-Kit](https://img.shields.io/github/sponsors/chindris-mihai-alexandru?label=Sponsor&logo=github)](https://github.com/sponsors/chindris-mihai-alexandru)
   
   Your sponsorship helps maintain and improve PWA-Kit for everyone.
   ```

6. **Publish your profile** and share on:
   - Twitter/X
   - Reddit r/macapps
   - Hacker News
   - Product Hunt
